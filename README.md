## Build

**REQUIREMENTS**

- Python

```
git clone https://github.com/minmin2104/yolo_obj_detection.git
cd yolo_obj_detection
python -m venv .venv
source .venv/bin/activate # Or on windows .\.venv\Script\activate
pip install -r requirements.txt
python main.py
```
